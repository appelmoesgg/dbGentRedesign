# Don Bosco Gent Redesign

In dit project verbeteren we de site van [Don Bosco Gent](https://donbosco.gent) , als eerste stap maken we
een statische site (een site die niet aangepast kan worden) en daarna gaan we een CMS (content managment system) bouwen
zodat scholen hun eigen info kunnen aanpassen!

## Statisch
> ✅ Voltooid!
We hebben de website volledig hermaakt met HTML, TailwindCSS en een klein beetje Javascript.
Op dit moment kun je hem [hier](https://donbosco-gent.be) vinden!
