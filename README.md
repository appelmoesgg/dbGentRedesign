# 🌟 Don Bosco Gent Website

![Don Bosco Gent Logo](https://donbosco-gent.be/wp-content/uploads/2020/03/logo.png)

Welkom bij het officiële GitHub-project van de **Don Bosco Gent** website! 🎓✨

---

## 📌 Over de Website
De website van **Don Bosco Gent** is ontworpen om:
- Ouders, leerlingen en bezoekers te informeren over ons onderwijs 🏫
- Belangrijke schoolinformatie en nieuws te delen 📰
- Online inschrijvingen en evenementen toegankelijk te maken 📅

🔗 **Website:** [donbosco-gent.be](https://donbosco-gent.be/)

---

## 🛠 Technologieën
Deze website is gebouwd met volgende webtechnologieën:

| Technologie | Functie |
|------------|---------|
| 🌐 **HTML** | Structuur van de pagina’s |
| 🎨 **CSS/tailwind** | Styling en layout |
| ⚡ **JavaScript** | dynamische elementen |
| 🖼 **Afbeeldingen** | Visuele content |
